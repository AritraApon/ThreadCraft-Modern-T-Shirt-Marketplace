# 👕 ThreadCraft – Modern T-Shirt Marketplace

ThreadCraft is a modern full-stack T-Shirt Marketplace built with **Next.js, TypeScript, MongoDB Atlas, and Better Auth**. The platform provides a seamless shopping experience for buyers while allowing sellers to securely manage their products through a dedicated dashboard. It features role-based authentication, responsive design, product management, analytics, and a clean user experience.

# ⚙️ Installation

### Clone Repository

```bash
git clone https://github.com/AritraApon/ThreadCraft-Modern-T-Shirt-Marketplace.git
```

### Navigate to Project Folder

```bash
cd ThreadCraft-Modern-T-Shirt-Marketplace
```

### Install Dependencies

```bash
npm install
```

### Configure Environment Variables

Create a `.env.local` file and add the following variables.

```env
MONGODB_URI=your_mongodb_uri

BETTER_AUTH_SECRET=your_secret

BETTER_AUTH_URL=http://localhost:3000

JWT_SECRET=your_jwt_secret
```

### Start Development Server

```bash
npm run dev
```
